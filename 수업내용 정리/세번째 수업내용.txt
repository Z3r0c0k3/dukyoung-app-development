안드로이드는 자바로 만들어졌다. 안배울수가 없다. 우린 이걸 배울거다.
우리가 만들 제품은 server-clint 서비스를 만들거임.
기본적인 구조로는 서버 단인 back-end와 클라이언트 단인 front-end가 있다.
# back-end(server)
    Web Application Service, DataBase Server 구성되어 있음.
# front-end(client)
    Client Application, client가 직접 보는 UI로 구성되어있음.
    inter/face의 대한 정의를 확실히 알자. (얼굴과 얼굴 사이에)
    우리가 만들 건 Client Application 이다.

내가 발전된점, 나를 명확히 나타낼수 있게 나를 소개해라.
개발이라는 것도 신념을 가지고 한다.
JAVA는 Object Oriented Programming (객체 지향 프로그래밍)의 신념를 가지고 있다.
# 객체 지향 프로그래밍이란?
    객체: 내가 떠올리는 모든 것에 대해 속성과 동작을 정의하는 것.
    객체 지향 프로그래밍이란 한마디로 내가 만들고 싶은걸 객체로 표현하는 것
    
    속성과 동작에 대해 정의가 끝났다면 구현을 해야한다.
    속성은 변수(상수 X), 동작은 함수로 구현한다.
    변수는 변하는 값을 저장하는 곳
    함수는 인풋 아웃풋

내가 상상하는걸 제품으로 만들어내기 위해 수업을 듣는것이다.
